部署教程 📤
How to put this online for free

推荐方案：GitHub Pages（免费）

  1. 注册一个 GitHub 账号
  2. 新建一个仓库（Repository），名字叫 你的用户名.github.io
  3. 把这个 HTML 文件改名为 index.html 上传到仓库
  4. 等 1-2 分钟，访问 https://你的用户名.github.io 就能看到了 ✿

关于照片

现在你的照片存在浏览器本地（IndexedDB），换电脑/清缓存会丢失，朋友也看不到。

方案 A · 最简单： 定期用工具栏的"↓ 导出 zip"备份到云盘。

方案 B · 真上线：

  1. 在 GitHub 仓库里创建 images/ 文件夹
  2. 把照片传进去（直接拖拽到 GitHub 网页就行）
  3. 编辑某条旅行时，"添加网络图片链接"那里填: images/你的图片名.jpg
  4. 这样朋友也能看到你的照片
